## 项目概述

BitPerfect —— 跨平台（macOS / Linux / Windows）本地音乐播放器，核心目标是绕过系统音频重采样，以独占模式（Hog Mode / WASAPI Exclusive / ALSA 直通）直出音频数据到 DAC。当前阶段仅开发 macOS 版本。

## 技术方向

- 语言：音频核心用 C++
